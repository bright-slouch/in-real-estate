# Quick Start Guide — re-open-house

5-minute overview for open house planning and follow-up.

---

## What This Skill Does

`re-open-house` manages three phases of the open house lifecycle:

**Pre-event** — 2-week promotion schedule with ready-to-post social content, an email blast, and property-specific talking points.

**Event day** — Sign-in sheet setup, conversation starters for different visitor types, and a seller security briefing checklist.

**Post-event** — A 3-touch follow-up sequence for every signed-in visitor plus a seller report.

---

## How to Invoke

```
/re-open-house [agent-slug]
```

Then describe what you need:

```
Plan the full open house for [address] — [date], [time]. [Key property features.]
```
```
Generate 2-week promotion schedule for my [city] listing open house — [date]
```
```
Write post-open-house follow-up for [N] visitors who signed in
```
```
Broker open invitation for [address] — [date], agents only
```
```
Seller report for open house at [address] — [N] visitors, feedback notes: [summary]
```

Your agent config loads automatically. Run `/re-agent-setup` first if you haven't set up your profile.

## 3 Most Common Uses

### 1. Full Open House Planning (Most Common)

Provide the property address, date/time, and top 3–5 features. The skill generates everything:

- 2-week promotion schedule with ready-to-post captions for each day
- Email blast copy for your sphere
- Property-specific talking points (not generic — specific to the features of this home)
- Sign-in sheet customized with the address and date
- Conversation starters for buyer, neighbor, and "already has an agent" visitor types
- Seller security briefing checklist

```
Full open house plan for [address] — [date, time], [beds/baths, price, school district, top features]
```

### 2. Post-Event Follow-Up Only

Already ran the open house and need the follow-up sequence?

```
Post-open-house follow-up for [address] — [N] visitors signed in. [Note any high-interest visitors by first name if helpful.]
```

The skill generates all 3 touches:
- Touch 1: Same-day thank you (text format for speed)
- Touch 2: 2-day check-in email with listing details
- Touch 3: 1-week follow-up for high-interest visitors (phone script + email version)

### 3. Seller Report

After the event, report to your seller the same evening or next morning:

```
Seller open house report for [address] — [N] visitors, [N] signed in. Feedback heard: [summary]. No offers yet.
```

The skill formats a professional seller report with traffic data, visitor profile, standout comments, any concerns raised, and your honest assessment of next steps.

## Sign-In Best Practices

**The sign-in sheet captures:** Name, email, phone, working with agent (Y/N), pre-approved (Y/N), timeline to buy, how they heard about the open house.

**How to ask:** Make it welcoming, not bureaucratic.
> "Would you mind signing in? It helps me send you the listing details after today."

Never pressure visitors. A visitor who doesn't sign in can still become a buyer — your job is to engage, not to enforce sign-in compliance.

---

## Open House Security — Remind Your Sellers

Before every open house, brief your sellers with the security checklist:

- Secure or remove: jewelry, cash, prescription medications, financial documents, passports, spare keys, small electronics, backup drives
- Lock rooms that won't be shown
- Plan to be away from the property during the event
- Do a walkthrough after to confirm nothing is missing

This conversation should happen at listing prep — not the morning of the open house.

---

## Follow-Up: The 80% That Actually Matters

Most of the open house value comes from the follow-up — not the event itself. The goal of each touch:

| Touch | Timing | Goal |
|---|---|---|
| Touch 1 | Same day (text) | Warm thank-you, low pressure, easy reply path |
| Touch 2 | 2 days later (email) | Provide value (listing details/brochure), surface urgency honestly if warranted |
| Touch 3 | 1 week later (phone/email) | Direct check-in for high-interest visitors only; open door to private showing |

**Never fabricate competing interest.** Touch 2 and Touch 3 should only reference other offers or serious activity if the seller and agent have confirmed it is real. Manufactured urgency damages trust and your reputation.
